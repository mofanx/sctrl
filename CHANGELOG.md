# 更新内容

## 1.0.0

### 核心功能
- 基于 Shizuku 的屏幕控制功能（熄屏/亮屏）
- HTTP API 服务器，支持远程控制屏幕
- 无障碍服务支持
- Compose UI 界面

### 技术特性
- 支持 arm64-v8a 和 x86_64 架构
- 最小 SDK 26 (Android 8.0)
- 目标 SDK 37
- Kotlin + Compose 技术栈

### 修复
- 修复 CI 构建中的 lint 检查问题
- 修复 JVM 选项兼容性问题
- 修复服务关闭时端口未释放的问题

### 优化
- 优化 Shizuku 授权体验
- 重构 UI/UX，优化启动屏幕
- 简化应用功能，专注屏幕控制
